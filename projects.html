Projects Portfolio

I have a number of personal and school projects I've worked on during my time here at George Mason. Below are some highlights. Most of these projects, along with other code blurbs, can be found on my Github account. The code for the projects below can also be found there.

Patriot 2048


Patriot 2048 is my fork of Doge 2048 by Jason LaFerrera, created off of the original 2048 by Gabriele Cirulli. It was made for fun. This project was created for some school spirit and consisted mainly of editing existing JavaScript and CSS.

The Force


This project was created by me and four other classmates for CS 101, and tied for the Best Project Award. The Force is an online remote designed for the iRobot Create. It communicates with a laptop on top of your iRobot, which sends commands to your robot. There is an integrated Skype stream on your webpage to see what your robot "sees." Once set up, the remote can be accessed from anywhere, allowing you to control your robot from anywhere in the world.

Spottag


This project was a Pebble prize-winner, created by me and two other classmates at HackNC 2014, a 24-hour hackathon at UNC Chapel Hill. SpotTag is a Pebble Watch application that allows the user to tag their current location and find their last tagged location relative to their current position. It was created using Simply.js.
